# PaleoAraripe

An educational board game about the paleontology of the Araripe Basin.

## About

A paleontological expedition through the Cretaceous of the Araripe Basin. Advance along the trail, identify fossils, and answer questions to reach the Paleontological Museum.

## How to play

- 🎲 Roll the dice to move along the trail
- 🪲 Ochre squares: identify the organism from a photo (up to 3 attempts)
- 📜 Green squares: answer questions about paleontology and taxonomy
- 🏛 Reach the Museum with the highest score possible

## Tech stack

- [TypeScript](https://www.typescriptlang.org/)
